# Тестовое задание
Тестовая задача для разработчика-стажёра в команду Информационной безопасности Авито для защиты сайта от ботов

## Быстрый старт

    $ docker-compose up

## Изменить стандартные интервалы и лимиты

Необходимо изменить файл **/.env**

    MASK=24 // префикс

    LIMIT=100 // лимит обращений за определенный интервал

    INTERVAL=60 // интервал в секундах

    TIMEOUT=120 // время бана подсети в секундах

Сохранение изменений

    # docker-compose up --build

## Тестирование

Предварительно установив все необходимые зависимости из **requirements.txt** в рабочее или виртуальное окружение, выполнить

    cd src/
    python tests.py
